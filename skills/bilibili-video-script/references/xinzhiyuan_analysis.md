# 新智元AIEra 视频脚本分析

> 基于35个热门视频分析总结

## 视频基本信息

- 时长：60-120秒（集中在90-110秒）
- 风格：快节奏科技资讯
- 更新频率：每天1-3个

## 标题公式

### 核心模式
```
[情绪词/数字] + [核心事件/产品] + [悬念/对比/数据]
```

### 具体模板

#### 1. 数字冲击型
- "AMD豪掷6GW+1.6亿股"
- "单季度681亿营收炸场"
- "2.4万假账号榨干Claude"

#### 2. 人物+事件型
- "老黄封神！"
- "奥特曼也签过"
- "五角大楼怒下最后通牒"

#### 3. 悬念提问型
- "为什么Meta却抢了头条？"
- "2028年ASI真要来？"
- "世界恐将迎来'幽灵GDP'？"

#### 4. 对比反差型
- "秒杀3万刀彭博终端"
- "直接把AI成本打穿"

## 描述（文案）公式

### 三段式结构
```
[核心事件] + [关键数据/细节] + [行业意义/影响]
```

### 示例

| 视频 | 描述 |
|------|------|
| AMD/Meta | Meta宣布部署6GW AMD GPU，绑定1.6亿股绩效认股权证。为何OpenAI没掀起浪？关键在承诺强度和执行确定性。 |
| 英伟达 | Q4营收681亿美元暴增73%，数据中心占比超91%。下一代Rubin系统推理成本直降10倍。AI进入"核电站级"时代。 |
| Perplexity | 集成19个大模型，多智能体协作，功能直指年费3万美元的Bloomberg。预示"人人皆CEO"时代。 |
| 五角大楼 | 国防部长限Anthropic周五前开放Claude军用，否则强制DPA。Grok已签协议入美军系统，AI军备战白热化。 |

## 内容结构（分镜）

### 30秒版本
- 开场 (0-5秒)：热点画面 + 核心亮点一句话
- 主体 (5-25秒)：事件细节 + 数据 + 简单分析
- 收尾 (25-30秒)：观点一句 + 引导互动

### 60秒版本
- 开场 (0-5秒)：热点切入 + 悬念
- 背景 (5-15秒)：事件详情 + 数据
- 分析 (15-45秒)：多角度解读 + 对比/案例
- 收尾 (45-60秒)：观点总结 + 行业展望

## 语言风格特点

| 特点 | 示例 |
|------|------|
| 简洁有力 | "老黄封神"、"炸裂" |
| 数据密集 | "6GW"、"681亿"、"73%" |
| 悬念感 | 问号结尾、疑问句 |
| 行业洞察 | 不仅说发生了什么，还说为什么重要 |
