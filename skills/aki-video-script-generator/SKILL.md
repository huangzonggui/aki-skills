# AI视频脚本生成器

基于新智元AIEra的短视频风格，生成30-120秒的AI科技资讯类视频脚本。

## 风格特点

**核心原则：开门见山、数据说话、悬念结尾**

### 标题公式
```
[核心事件/人物] + [数字/数据] + [悬念/反问]
```

**示例：**
- AMD豪掷6GW+1.6亿股！奥特曼也签过，为什么Meta却抢了头条？
- 老黄封神！单季度681亿营收炸场，Rubin直接把AI成本打穿

### 脚本结构（60秒版本）

```
[0-5秒] 开门见山：抛出核心事件/数据
[5-15秒] 背景交代：为什么发生
[15-25秒] 核心细节：关键数据/人物
[25-35秒] 观点/分析：行业影响
[最后]    悬念结尾 or 升华总结
```

## 使用方法

1. **用户提供素材**：新闻热点、事件描述、关键数据
2. **选择模板**：热点解读型 / 数字冲击型 / 人物动态型
3. **生成脚本**：自动填充内容

## 参考资料

详细公式和模板见：`references/视频脚本公式.md`

## 输出格式

生成内容包括：
1. **标题**（2-3个选项）
2. **脚本正文**（60秒/90秒/120秒版本）
3. **分镜建议**（画面描述+时长）
4. **制作 Checklist**

**输出：**
