# 视频脚本公式详解

## 核心原则

**前5秒固定，后面灵活**

```
[核心事件/人物] + [数字/数据] + [悬念/反问]
```

→ 开门见山，直接抓注意力。后面的内容根据时长灵活展开（30秒/60秒/120秒均可）。

---

## 一、三大模板

### 模板1：热点解读型

**结构：** 事件 + 数据 + 为什么重要

**适用场景：** 重大行业新闻、公司财报、技术突破

**公式：**
```
前5秒：[核心事件/人物] + [数字/数据] + [悬念/反问]

后面：
- 背景交代
- 核心数据展开
- 行业影响分析
- 悬念结尾/互动
```

**案例：**
> 前5秒：就在昨天，Meta宣布部署6GW AMD GPU，还绑定了1.6亿股绩效认股权证！这意味着什么？
> 
> 同样结构，AMD去年也给过OpenAI。为什么OpenAI没掀起浪，Meta却震动市场？ 
> 
> 关键在承诺强度、算力主导权与执行确定性。 
> 
> 你觉得这是AI算力大战的转折点吗？

---

### 模板2：数字冲击型

**结构：** 数字 + 对比 + 震撼结果

**适用场景：** 财报数据、融资消息、排名发布

**公式：**
```
前5秒：[大数字]！这是[公司]的[记录]！到底什么概念？

后面：
- 过去vs现在对比
- 行业影响分析
- 未来预测
```

**案例：**
> 前5秒：681亿！英伟达Q4营收炸裂，同比增长73%！这是什么概念？
> 
> 数据中心收入占比超过91%，相当于再造一个AMD。 
> 
> 更狠的是下一代Rubin系统：HBM4、液冷架构，推理成本直降10倍。 
> 
> AI基础设施进入"核电站级"时代。

---

### 模板3：人物动态型

**结构：** 人物 + 动作 + 大动作

**适用场景：** 巨头动向、明星创始人、重要任命

**公式：**
```
前5秒：[人物]又出[动作]了！这次[具体内容]，到底有多[结果]？

后面：
- 细节展开
- 技术/行业意义
- 未来展望
```

**案例：**
> 前5秒：马斯克放出特斯拉Optimus独舞视频！这次单脚平衡，到底有多强？
> 
> 单脚平衡能力卓越，动作自然流畅，比人类还丝滑。 
> 
> 这意味着[人形机器人技术突破]。 
> 
> 2026年，我们或迎来机器人元年？

## 二、标题公式

### 核心公式
```
[核心事件/人物] + [数字/数据] + [悬念/反问]
```

### 标题词汇库

**动词（动作）：**
- 豪掷、宣布、放出、发布、收购、任命
- 炸场、封神、突破、崛起、称霸、终结

**修饰词：**
- 震撼、炸裂、疯狂、硬核、劲爆
- 史上、首次、突破性、里程碑

**悬念词：**
- 为什么？真的吗？会怎样？来了？
- 真相是...、内幕是...、结果却是...

## 三、分镜指南

### 通用结构（前5秒固定）

| 时段 | 秒数 | 内容 |
|------|------|------|
| 开场 | 0-5 | **固定**：[核心事件] + [数字] + [悬念] |
| 展开 | 5- | 灵活展开（背景/数据/分析/结尾） |

### 30秒版本

| 时段 | 秒数 | 内容 | 画面 |
|------|------|------|------|
| 开场 | 0-5 | **固定公式** | 热点关键词大字报 |
| 事件 | 5-15 | 核心事件 | 简报/新闻画面 |
| 数据 | 15-25 | 关键数字 | 数字动画放大 |
| 分析 | 25-28 | 观点输出 | 图文卡片 |
| 结尾 | 28-30 | 悬念/互动 | 问句+引导关注 |

### 60秒版本

| 时段 | 秒数 | 内容 | 画面 |
|------|------|------|------|
| 开场 | 0-5 | **固定公式** | 热点话题切入 |
| 背景 | 5-20 | 事件背景 | Timeline/简报 |
| 核心 | 20-45 | 数据/细节 | 多图表切换 |
| 分析 | 45-55 | 观点分析 | 图文卡片 |
| 结尾 | 55-60 | 总结/悬念 | 开放性问题 |

## 四、制作要点

### 必须有
- **前5秒固定公式**（0-5秒必须用）
- 具体数据支撑
- 明确观点/分析
- 结尾悬念或引导

### 不要有
- 废话铺垫（前5秒后再说背景）
- 含糊描述
- 中立无观点
- 平淡结尾

### 节奏把控
- 前5秒必须抓住注意力
- 每5秒一个信息点
- 数字要放大强调
- 语速适中偏快
- 画面信息密集
