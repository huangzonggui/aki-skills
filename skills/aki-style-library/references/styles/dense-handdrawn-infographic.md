# dense-handdrawn-infographic

High information density, colorful hand-drawn infographic cover. Clean but lively, clear hierarchy, rich details.

## Prompt Template (CN)

```
高信息密度彩绘手绘信息图封面，3:4 竖版，米白纸纹背景，细腻线稿+水彩上色，色彩丰富且协调。

主题：{主题}
主标题：{主标题}
副标题：{副标题}

核心构图：
- 画面中心：{中心主体}，手持或连接一个“代码/流程面板”。
- 环绕流程：用箭头/虚线连接 5 个步骤节点，节点有编号与小字标签：{步骤1} → {步骤2} → {步骤3} → {步骤4} → {步骤5}。
- 小图标元素：云端代码、skills 积木、浏览器窗口、剪贴板、图片缩略图、微信编辑器、CDP/链路、齿轮与连接线。
- 信息密度高但结构清晰，层级分明，文字可读。

配色：
- 主色：#d93630（红色用于关键节点与标题）
- 背景：温暖米白/纸张纹理
- 强调色：青绿、橙色、柔蓝

风格要求：
- 手绘/水彩/彩绘风格，干净利落但细节丰富。
- 类信息图布局，流程图+模块化面板，整体专业、清晰、视觉密集。

负面词：写实照片、3D渲染、暗色主题、低细节、模糊、杂乱、logo、水印、英文大段文字。
```

## Usage Notes

- 适合高信息密度封面、流程可视化、工具链示意图。
- 标题尽量短（4-8 字），副标题一行即可。
- 流程节点建议 4-6 个，避免拥挤。
