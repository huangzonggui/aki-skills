# handnote

Minimal hand-drawn note, logical and clean

Source of truth: `skills/aki-style-library/references/styles/手绘逻辑信息艺术设计师.md`

## Color Palette

- Primary: Black (#1A1A1A)
- Background: Pure white (#FFFFFF)
- Accents: Red (#E53E3E), green (#48BB78), yellow (#F6E05E) for highlights

## Visual Elements

- Thin rounded hand-drawn lines, marker highlights
- Simple icons and flow diagram shapes
- No gradients, shadows, photos, or 3D

## Typography

- Oversized hand-lettered title
- Smaller handwritten labels for secondary nodes

## Layout

- Clear hierarchy with whitespace
- Avoid overlapping arrows; keep each card concise

## Best For

Logic maps, summaries, process explanations
